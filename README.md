Este é um projeto de Gestão de Finanças Pessoais desenvolvido com o objetivo principal de praticar a integração de diversas bibliotecas no ecossistema Node.js e a comunicação entre o front-end e o back-end.

O foco aqui não é apenas a funcionalidade, mas o domínio das ferramentas e o entendimento de como cada biblioteca contribui para uma aplicação robusta.

🚀 Tecnologias e Bibliotecas
O projeto utiliza as seguintes tecnologias:

Back-end
Node.js: Ambiente de execução Javascript.

Express: Framework para construção das rotas e APIs.

MySQL2: Driver para conexão e execução de queries no banco de dados MySQL.

CORS: Gerenciamento de permissões para requisições externas.

Dotenv: Proteção de dados sensíveis (senhas e chaves) através de variáveis de ambiente.

Nodemon: Ferramenta de auxílio ao desenvolvimento que reinicia o servidor automaticamente.

Path: Manipulação de caminhos de arquivos e diretórios.

Front-end
HTML5 & CSS3: Estruturação e estilização da interface do usuário.
